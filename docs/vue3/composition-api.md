# Composition API 相关

## Composition API 优势

- 更好的代码组织：可以将相关的逻辑组织在一起，提高代码的可读性和可维护性。
- 更好的类型推断：对 TypeScript 支持更好，提供了更清晰的类型定义。
- 更好的代码复用：可以通过组合函数复用逻辑，而不是通过 mixins 或 HOC。
- 更灵活的响应式系统：可以在任何地方创建响应式数据，而不仅限于组件选项。
- 更好的性能：减少了选项 API 中的性能开销，例如不需要初始化所有选项。

## script setup 原理

- script setup 是 Vue3 的语法糖，用于简化 Composition API 的使用。
- 它会自动将顶层变量和函数暴露给模板，无需 return。
- 它会自动处理组件的导入和注册，无需手动注册组件。
- 它会在编译时将代码转换为 setup 函数，执行时机与 setup 相同。
