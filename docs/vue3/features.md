# 其他特性相关

## Teleport 作用

- Teleport 是 Vue3 的内置组件，用于将组件的内容渲染到指定的 DOM 节点。
- 它可以将组件内容从当前组件树中移动到指定的 DOM 位置，而不影响组件的逻辑关系。
- 适用于模态框、通知、弹出菜单等需要脱离父组件容器的场景。

## Suspense 原理

- Suspense 是 Vue3 的内置组件，用于处理异步组件的加载状态。
- 它可以在异步组件加载完成前显示一个 fallback 内容，加载完成后显示组件内容。
- 适用于需要加载异步数据或异步组件的场景，提供更好的用户体验。

## Fragment

- Fragment 是 Vue3 的特性，允许组件返回多个根节点，而不需要包裹在一个父元素中。
- 它可以减少 DOM 节点数量，提高渲染性能。
- 适用于需要返回多个元素的组件，例如列表项、表格行等。
